spritesheet module¶

class spritesheet.SpriteSheet(filepath)¶

Bases: pygame.sprite.Sprite

Constructor method, creates a Sprite sheet object used for tile creation

Parameters

filepath – the path for the tiles sprite sheet that must have the tiles places in line and at equal distance set by the constant TILE_GAP

all_sprites(mahjong_tiles_dictionary, middle_x, middle_y)¶

Function responsible for creating a list of Tile objects by reading a dictionary with tuples of information

Parameters

  • mahjong_tiles_dictionary – a dictionary with a unique identifier that has attached the relative coordinates, tile layer and tile type in a tuple

  • middle_x – pygame.screen middle x

  • middle_y – pygame.screen middle y

Returns

list of tile.Tile()

sprite_at(identifier, sprite_sheet_x, sprite_sheet_y, tile_width, tile_height, on_screen_x, on_screen_y, layer, tile_type)¶

Function responsible for creating a singular Tile object with specific zoom factor, color correction and surface and rectangle grouping

Parameters

  • identifier – unique number for identifying tiles

  • sprite_sheet_x – the x coordinate in the sprite sheet

  • sprite_sheet_y – the y coordinate in the sprite sheet

  • tile_width – the width of the tile

  • tile_height – the height of the tile

  • on_screen_x – the absolute x position of the tile on screen

  • on_screen_y – the absolute y position of the tile on screen

  • layer – the height attribute of the tile object, used for interractions

  • tile_type – the type of the tile object, specifies the relative position of the tile in the sprite sheet and the the types that can match with it

Returns

tile.Tile()
