initialize module¶

class initialize.Initialize(mahjong_dictionary)¶

Bases: object

Class used for easy level creation, working for any zoom level as long as the screen is big enough; the constructor has a counter, an empty dictionary and a list with all the tiles type. It can hold up to a maximum of 152 tiles!

Parameters

mahjong_dictionary – empty or partially filled dictionary

Type

dict

draw_line(start, end, static_coord, layer_height, orientation)¶

Draws a line of tiles equally distanced apart, between start and end, at a fixed coordinate with the option of changing the direction and layer height

Parameters

  • start (float) – a float number that can take any value

  • end (float) – a float number that can only take values equal to start+2n

  • static_coord (float) – the fixed coordinate for the line to appear at

  • layer_height (int) – the height of the tile line, the bigger the number the higher the line; affects interactions

  • orientation (str) – can either be vertical or horizontal, any other value will return an unmodified dictionary

Returns

the modified dictionary that can be represented as a line of tiles

Return type

dict

draw_piece(coord_x, coord_y, layer_height)¶

Draws a tile at a specific x and y and height

Parameters

  • coord_x (float) – the middle x coordinate of the tile, recommended between -13 and 14

  • coord_y (float) – the middle x coordinate of the tile, recommended values between -3.5 and 8.5

  • layer_height (int) – used for determining if a tile is over, under or around another tile

Returns

updates the dictionary
