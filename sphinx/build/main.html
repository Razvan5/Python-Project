main module¶

main.check_unwinnable_game(tile_sprite_group)¶

Checks every tile for potential pairings and if no pairings are possible for all tiles on the screen

Parameters

tile_sprite_group – the tile group

Type

pygame.sprite.Group()

Returns

returns if the game is unwinnable

main.init_level()¶

initializes a level using the initialize.Initialize() module, overwrites the global mahjong dictionary

main.initialize_game()¶

Initializes a tuple of variables used by the main while loop to perform certain actions

Returns

a tuple made of a background surface, a cursor.Cursor() object, a cursor sprite.Group(), a tile

sprite.Group() and a boolean variable to decide the game state

main.initialize_tiles()¶

The function uses the global mahjong dictionary to construct pygame.sprite.Group() with the properties found in the dictionary

Returns

a sprite group containing all the tiles to be rendered and interacted with

Return type

pygame.sprite.Group()

main.order_dictionary()¶

The function orders the tiles in the tile dictionary so the sprites are rendered in a correct 3D fashion with little to no overlaps

main.render_text(text_group, text_location, text_shadow_location)¶

Renders different text groups on screen for less game loop cluttering

Parameters

  • text_group – a tuple made out of the text surface and text shadow surface

  • text_location (tuple of ints) – the texts locations

  • text_shadow_location (tuple of ints) – the location of the text’s shadow

main.shuffle()¶

Shuffles the tile types from the tiles dictionary, a global variable
